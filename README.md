<h2>Usman Zaman</h2>

<p>A clean, responsive portfolio showcasing front-end development projects, modern UI design, and interactive web experiences.</p>

<h2>Tech Stack</h2>

<ul>
  <li>HTML.</li>
  <li>CSS.</li>
  <li>JavaScript.</li>
  <li>Bootstrap.</li>
  <li>Figma.</li>
  <li>Visual Studio Code.</li>
</ul>

<h2>Features</h2>

<ul>
  <li>Light/Dark theme toggle with saved user preference and automatic system theme detection.</li>
  <li>Fully responsive design with touch-optimised navigation and mobile-first interactions.</li>
  <li>Smooth scrolling, animated transitions, and dynamic active navigation highlighting.</li>
  <li>Interactive project showcase featuring autoplay videos, hover previews, and mobile video support.</li>
  <li>Functional contact form with live validation, submission feedback, and Formspree integration.</li>
  <li>Optimised loading experience with a custom preloader, adaptive animations, and scroll restoration handling.</li>
  <li>Responsive testimonials and content sliders powered by reusable carousel components.</li>
  <li>Interactive FAQ section with animated expandable panels and responsive behaviour.</li>
  <li>Polished user experience with magnetic navigation effects, shrinking header, and animated scroll indicators.</li>
  <li>Performance-focused enhancements, including conditional desktop animations and responsive media optimisation.</li>
</ul>
