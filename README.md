<h1>Usman Zaman Portfolio</h1>

<p>
A modern portfolio built with Next.js, React, and TypeScript, showcasing responsive web applications, interactive user experiences, and front-end development projects.
</p>

<h2>Tech Stack</h2>

<ul>
  <li>HTML5.</li>
  <li>CSS3.</li>
  <li>SCSS.</li>
  <li>TypeScript.</li>
  <li>React.</li>
  <li>Next.js.</li>
  <li>Git & GitHub.</li>
  <li>Figma.</li>
  <li>Visual Studio Code.</li>
</ul>

<h2>Features</h2>

<ul>
  <li>Responsive, mobile-first design optimised for desktop, tablet, and mobile devices.</li>
  <li>Light and dark themes with persistent user preference and smooth transitions.</li>
  <li>Smooth scrolling, interactive animations, and active navigation highlighting.</li>
  <li>Project showcase featuring responsive video previews and hover interactions.</li>
  <li>Reusable React component architecture built with Next.js and TypeScript.</li>
  <li>Performance optimisations including image optimisation and modern loading techniques.</li>
  <li>Custom preloader with seamless page transitions and scroll restoration.</li>
  <li>Accessible contact form with client-side validation and responsive feedback.</li>
  <li>SEO-ready setup with metadata, robots.txt, sitemap.xml, and structured data.</li>
  <li>Clean, scalable codebase using modular components and organised styling architecture.</li>
</ul>
